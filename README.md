## 🚀 快速开始

### 系统要求
- **Node.js** >= 18.0
- **PostgreSQL** >= 12.0
- **npm** 或 **pnpm**

### 安装部署

```bash
# 克隆项目
git clone https://github.com/your-org/dental-ai-pro.git
cd dental-ai-pro

# 安装依赖
npm install
# 或使用 pnpm（推荐）
pnpm install

# 配置环境变量
cp .env.example .env
# 编辑 .env 文件，配置数据库等信息

# 数据库迁移
npm run db:migrate

# 启动开发服务器
npm run dev
```

### 访问系统
- **前端应用**：http://localhost:5174
- **API文档**：http://localhost:3001/api/docs
- **进化仪表板**：http://localhost:5174/evolution
- **项目展示**：http://localhost:5174/pitchdeck
